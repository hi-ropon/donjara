麻雀牌画像フォルダ

このフォルダには麻雀牌の画像ファイルを配置してください。

フォルダ構成:
- wan/    : 萬子（1.png から 9.png）
- pin/    : 筒子（1.png から 9.png） 
- sou/    : 索子（1.png から 9.png）
- honor/  : 字牌（1.png から 7.png）
  - 1: 東
  - 2: 南  
  - 3: 西
  - 4: 北
  - 5: 白
  - 6: 發
  - 7: 中

画像ファイル形式: PNG推奨（JPG, GIF, BMP も対応）
画像サイズ: 推奨 40x60 ピクセル（自動リサイズされます）

注意: 画像ファイルが存在しない場合は、文字表示になります。
